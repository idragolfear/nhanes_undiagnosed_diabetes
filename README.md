# Undiagnosed Diabetes in NHANES 2017–2018

This project analyzes the prevalence of undiagnosed diabetes among adults
using NHANES 2017–2018 data, with survey-weighted methods.

## Data
Data source: National Health and Nutrition Examination Survey (NHANES) 2017–2018.

## Methods
Cross-sectional analysis with survey weights.
Main outcome: undiagnosed diabetes defined by HbA1c ≥ 6.5% without self-reported diagnosis.

## Reproducibility
Run the scripts in the `analysis/` folder to reproduce the results.
